About the System

Intelligent Trainer is designed as a hybrid intelligent decision support system for individualized training recommendations.

Machine Learning Module

The system predicts the user’s training intensity level using physiological and workout-related indicators.

Recommendation Engine

A rule-based layer adapts the workout plan according to user goal, focus area, and equipment availability.

Academic Purpose

The project demonstrates how physiological data can be combined with AI techniques for personalized workout recommendation.

Future Expansion

Future versions may include workout logging, progress analytics, exercise videos, and adaptive recommendation updates.
